# Panel Outline and Visual System

## Current flow

1. **Introduction**
   - Title slide with direct-to-student framing.
   - Slide-style left/right navigation.

2. **Example trial**
   - Fixation cross.
   - Circular sine-wave grating.
   - `F` and `J` response keys.
   - Green or red feedback ring around the stimulus.

3. **Demo task**
   - Short hidden-rule sorting task.
   - No stimulus-space explanation yet.
   - One-dimensional x structure under the hood, but introduced only as a demo game.

4. **Where do stimuli come from?**
   - Generic 2D stimulus space with no boundaries.
   - Hover inspection connects points to gratings.

5. **Uni-x**
   - Vertical boundary on the left.
   - Task and trial flow on the right.
   - Current stimulus appears in the space and becomes coded by response and feedback.

6. **Uni-y**
   - Horizontal boundary on the left.
   - Same task display on the right.

7. **II**
   - Diagonal boundary on the left.
   - Same task display on the right.

## Visual system

- Dark scientific-editorial background.
- Serif display headings plus sans-serif instructional text.
- Large circular gratings with clean masks.
- Slide-sized panels with fixed side navigation.
- Teaching copy addressed directly to the learner.
