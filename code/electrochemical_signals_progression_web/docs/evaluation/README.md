# Passive Signalling Simulation Evaluation Kit

This folder contains a lightweight evaluation and documentation package for the passive signalling simulations in this project.

## Files

- `concept_inventory.md`: 10-item pre/post concept inventory with answer key and scoring notes
- `instructor_observation_sheet.md`: structured classroom observation form
- `student_feedback_form.md`: short learner feedback form
- `semester_results_template.md`: semester-level reporting template for documenting outcomes

## Recommended Use

Use the materials in four stages:

1. Before teaching
   - Record the cohort, course context, prior exposure, and intended learning outcomes.
   - Administer the concept inventory as a pre-test or use a subset of items.
2. During teaching
   - Use the instructor observation sheet while students interact with the simulations.
   - If possible, collect brief notes from tutors as well as the lead instructor.
3. Immediately after teaching
   - Administer the same concept inventory as a post-test.
   - Administer the student feedback form.
4. After the module
   - Complete the semester results template.
   - Archive sample student explanations, common misconceptions, and any planned revisions.

## Suggested Evidence Package

For each semester, aim to keep:

- Pre/post concept inventory scores
- A copy of the teaching activity or worksheet used with the simulations
- Completed instructor observation sheets
- Student feedback summaries
- A brief narrative of what worked, what did not, and what will change next time

## Practical Notes

- The concept inventory is designed around the sequence in this project:
  - diffusion
  - membrane pores and permeability
  - electrochemical drift
  - resting potential
  - Nernst equilibrium
  - Goldman-style multi-ion reasoning
- You can use the full inventory across the whole sequence or split it into topic blocks.
- If you want stronger evidence, add a delayed retention quiz 1-2 weeks later using 3-5 of the same items.
