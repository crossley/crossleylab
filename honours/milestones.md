# Milestones

Generic honours project milestone sequence (date-free evergreen version).

## Sequence

1. Decide on project
2. Define rationale, methods, and predicted results
3. Submit ethics / obtain ethics approval
4. Finalize experiment programming and materials
5. Participant recruitment / data collection
6. Draft methods
7. Finish data collection / draft introduction
8. Data analysis
9. Draft results
10. Draft discussion
11. Thesis submission

## Usage

Keep actual dates on the cohort page and link back here for the generic sequence.
